Camork 개인정보 처리방침

최종 업데이트: 2026년 5월 15일

1. 수집하는 정보

Camork는 사용자의 사진, 동영상, 메모, 촬영 위치 정보를 사용자 기기 내에서만 저장합니다. 어떠한 개인 데이터도 외부 서버로 전송되지 않으며, 개발자를 포함한 제3자가 접근할 수 없습니다.

2. 카메라 및 마이크

카메라 접근은 사진·동영상 촬영 목적으로만 사용되며, 촬영된 미디어는 앱 내 로컬 저장소에만 보관됩니다. 마이크는 동영상 녹화 시 오디오 수집을 위해 사용됩니다.

3. 위치 정보

위치 정보는 촬영 장소를 사진에 기록하기 위한 목적으로만 사용됩니다. GPS 데이터는 촬영된 미디어 파일의 메타데이터로 기기 내에만 저장됩니다.

4. Face ID / 생체 인증

Face ID 등 생체 인증 정보는 Apple의 LocalAuthentication 프레임워크를 통해 처리되며, Camork 앱은 생체 데이터 자체에 접근하지 않습니다. 인증 성공·실패 여부만 확인합니다.

5. 데이터 보관 및 삭제

모든 데이터는 사용자 기기 내에만 존재합니다. 앱을 삭제하면 관련된 모든 사진·동영상·메타데이터가 함께 삭제됩니다. 앱 내 휴지통 기능을 통해 삭제된 항목은 30일 후 자동으로 영구 삭제됩니다.

6. 제3자 공유

Camork는 사용자 데이터를 수집하지 않으므로 제3자와 공유할 데이터가 존재하지 않습니다. 광고 목적의 데이터 수집도 하지 않습니다.
