Fanlo 개인정보 처리방침

최종 업데이트: 2026년 5월 15일

1. 수집하는 정보

Fanlo는 사용자가 앱에 직접 입력한 연락처, 고객 메모, 방문 기록, 샘플 배포 기록, 일정 및 관련 업무 데이터를 사용자 기기 내에 저장합니다. 해당 데이터는 앱 기능 제공을 위해서만 사용됩니다.

2. 연락처 접근

사용자가 권한을 허용한 경우 Fanlo는 연락처를 불러와 고객 관리 기능에 활용할 수 있습니다. 연락처 데이터는 사용자의 기기 내에서 처리되며, 개발자 서버로 전송되지 않습니다.

3. 알림

Fanlo는 방문 일정, 후속 작업, 고객 관리 알림을 제공하기 위해 로컬 알림 권한을 요청할 수 있습니다. 알림 데이터는 기기 내에서 관리됩니다.

4. 데이터 보관 및 삭제

사용자가 입력한 데이터는 앱이 설치된 기기 내에 보관됩니다. 앱 내 삭제 기능 또는 앱 삭제를 통해 관련 데이터를 삭제할 수 있습니다.

5. 제3자 공유

Fanlo는 사용자의 개인 데이터를 판매하거나 광고 목적으로 공유하지 않습니다. 사용자가 명시적으로 내보내기 또는 공유 기능을 실행하는 경우에만 선택한 데이터가 사용자가 지정한 대상으로 전달됩니다.
